# 🩺 MedVerify AI

## 🚀 Overview
MedVerify AI is a web-based solution designed to:
1. Classify clinical documents and check compliance with Standard Treatment Guidelines (STG)
2. Detect potential document forgery or manipulation

---

## 🌐 Live Demo
👉 https://medverify-ai-tkpeli3aatbgdhjcwxtgtl.streamlit.app/

---

## ✨ Features
- 📄 Upload medical documents
- 🧠 Automatic document classification
- ✅ STG compliance detection
- ⚠️ Forgery probability scoring
- 💬 Chatbot explanation system

---

## 🛠️ Tech Stack
- Python
- Streamlit
- FastAPI (initial backend)
- GitHub

---

## 🧪 How It Works
- Text is extracted from uploaded document
- Keywords are used to classify document type
- Rules detect compliance issues
- Randomized anomaly scoring simulates forgery detection
